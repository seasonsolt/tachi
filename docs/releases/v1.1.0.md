# Tachi v1.1.0 Release Notes

Tachi is now the primary macOS companion identity for the project.

## Highlights

- Renamed the macOS app, executable, bundle identity, and launch agent to Tachi.
- Added the Tachi app icon and refreshed README product presentation.
- Restored reliable Codex quota display, including reset count and reset valid dates.
- Fixed session monitoring stalls caused by large process-list output.
- Added provider-backed session monitoring for Claude Code, Codex, OpenCode, and Pencil.
- Removed setup-first flow assumptions so Tachi watches standard local tool traces by default.
- Refined companion/theme behavior so the menu bar state follows the active companion theme.

## Artifact

- `Tachi-1.1.0-macos-unsigned.zip`

This is an unsigned preview build. Developer ID signing and notarization are still required before public macOS distribution.
